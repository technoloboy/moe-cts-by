Boying 与 Go2 课程学习完整对比

  ---
  一、地形难度课程（共同）
  
  函数：terrain_levels_vel_gym，在每个 episode reset 时评估

  升降规则：
  - 升级：该 episode 内 max_move_dist > terrain_length/2 = 4m → 进入更难的行（更高 level）
  - 降级：max_move_dist < 目标距离 × 0.5 且未升级 → 退回更容易的行
  - 地形共 10 行（level 0~9），初始分布在 0~5 行（max_init_terrain_level=5）

  两者完全相同，地形类型和高度参数有差异（Boying 台阶最高 15cm，Go2 25.7cm）。

  ---
  二、奖励权重渐变课程（gradual_reward_weight_modification）
  
  ┌──────────────────────┬────────────────┬────────────────────────┬─────────────────────────┐
  │        课程名        │  控制的奖励项  │         Boying         │           Go2           │
  ├──────────────────────┼────────────────┼────────────────────────┼─────────────────────────┤
  │ base_linear_velocity │ lin_vel_z_l2   │ -2.0 → 0.0，it=0~1500  │ -2.0 → 0.0，it=0~1500   │
  ├──────────────────────┼────────────────┼────────────────────────┼─────────────────────────┤
  │ base_height_l2       │ base_height_l2 │ -1.0 → -8.5，it=0~7000 │ -1.0 → -10.0，it=0~5000 │
  └──────────────────────┴────────────────┴────────────────────────┴─────────────────────────┘

  lin_vel_z_l2 课程含义：训练早期用 -2.0 强力抑制垂直弹跳，帮助机器人建立平衡；1500 iter 后权重归零，交由 base_height_l2 接管高度管理。

  base_height_l2 课程含义：初期权重小（-1.0），避免高度惩罚与早期探索冲突；逐渐加强，最终 Boying 用 -8.5（Go2 用 -10.0，差异来自机械参数比例缩放）。

  ---
  四、指令速度课程的附加机制
  
  零命令概率课程（zero_command_curriculum）：

  it=0~1500 线性增长，最终零命令概率 = 10%（两者相同）。训练过程中每个 episode 有 10% 概率速度命令为零，要求机器人学会原地静止站立。

  地形自适应命令限制（terrain_max_command_ranges）：

  根据机器人当前所在地形类型，自动限制最大命令速度（Boying 和 Go2 相同）：

  ┌────────────────────────┬───────────┬─────────────┐
  │        地形类型        │ lin_vel_x │ ang_vel_yaw │
  ├────────────────────────┼───────────┼─────────────┤
  │ flat                   │ ±2.0 m/s  │ ±2.0 rad/s  │
  ├────────────────────────┼───────────┼─────────────┤
  │ wave/slope/rough_slope │ ±1.5 m/s  │ ±1.5 rad/s  │
  ├────────────────────────┼───────────┼─────────────┤
  │ stairs/obstacles/gap   │ ±1.0 m/s  │ ±1.5 rad/s  │
  └────────────────────────┴───────────┴─────────────┘

  极限速度采样（limit_vel_prob=0.2）：

  每次重采样命令时，有 20% 概率直接从 limit_vel = {[-1,1], [-1,1], [-1,0,1]} 中采样最大值（±最大速度或零），强制训练极端工况。

  ---
  五、课程学习时间线总览
  
  Boying:
  it=    0: 开始训练，命令 ±0.5 m/s，垂直速度惩罚 -2.0，高度惩罚 -1.0
  it= 1500: lin_vel_z_l2 权重归零（垂直弹跳约束解除）
  it= 7000: base_height_l2 权重达到最终值 -8.5
  it=50000: 命令扩展到 ±1.0 m/s
  it=100000: 命令扩展到 ±2.0 m/s（最大）
  地形升降: 全程动态，由机器人移动距离决定

  Go2:
  it=    0: 同上
  it= 1500: lin_vel_z_l2 归零（同 Boying）
  it= 5000: base_height_l2 达到最终值 -10.0（比 Boying 更快更强）
  it=20000: 命令扩展到 ±1.0 m/s（比 Boying 早 30k）
  it=50000: 命令扩展到 ±2.0 m/s（比 Boying 早 50k）
